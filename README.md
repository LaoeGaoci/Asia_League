# 亚洲联盟与领袖能力

## 1. 亚洲联盟特性

### 💴 藏富于民
- 每位公民提供 +4 金币加成。

### 🫂 民族共和
- 每条商路提供 +5 粮食、+5 金币和 +5 信仰值。

## 2. 亚洲领袖特性

### 天下大同
- 提高 10% 奇观生产力，让建造者可以加速奇观建造 15%。
- 外交友好（存在部分问题无法测试）。

## 3. 特色建筑

### 🏫 亚洲国立学术中心
- **解锁时间**：写作 `TECH_WRITING`
- 自身无科技值加成。
- 每回合提供 3 点大科学家点数。
- 为相邻 1 格学院提供 +2 的科技值加成。
- 可以修建多所。

### 🏦 亚洲金融中心
- **解锁时间**：货币 `TECH_CURRENCY`
- 自身无金币加成。
- 每回合提供 5 点大商人点数。
- 为相邻 1 格商业中心提供 +10 的金币加成。
- 可以修建多所。

### 🏯 亚洲国立文化中心
- **解锁时间**：诗歌与剧院 `CIVIC_DRAMA_POETRY`
- 自身无文化值加成。
- 每回合提供 3 点大艺术家点数。
- 为相邻 1 格剧场提供 +2 的文化值加成。
- 可以修建多所。

### 1. [如何定义一个文明](./doc/Civilization/Define_Asia.md)
- 确定文明名称与背景。

- 设定领袖名称与背景。
- 定义领袖能力，包括特色加成、外交倾向等。
- 绑定该领袖到对应的文明。

- 确定区域名称及解锁条件。
-  [设定区域的基础效果与相邻加成](./doc/DISTRICT/District_addition.md)。
- 在代码中定义区域数据。
